# Tourist Vista (Frontend Prototype)

A static single-page travel UI built with **HTML**, **CSS**, and **vanilla JavaScript**.

## Features

- Hero section with background video
- Search UI (UI-only) and destination filtering
- Featured destination cards with hover effects
- Destination details modal (gallery, weather, attractions)
- Travel planner form with validation and result card
- Gallery grid with lightbox popups
- Contact form with validation
- Dark mode toggle + smooth scrolling + responsive design

## How to run

1. Open `index.html` in your browser.
2. No backend required.

## Notes

- All data is static and stored in `script.js`.
- No APIs or server logic are used.
